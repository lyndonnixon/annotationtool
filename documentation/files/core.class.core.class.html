Package: Core
Core
This class provides the core functions of the ConnectME hypervideo annotation tool. (Main Class!) Those are: * Parse and load the settings file * Parse and load the language file * Parse the opened video and generate a video object out of it, which contains the video ID and locators * Parse the administration settings and generate an object of the selected CMF details (URL, access data, protected URL) * Parse the saving currenlty logged in user OpenID

getCMF() : array

Get CMF detials

Tries to parse administration settings to get the selected CMF instance. The default CMF details of the settings file will get used, if no administration settings file is available. Furthemore, it tries to generate a protected URL which includes username and password for accessing protected CMF functions like sparql update. CMF definition file is located at /core/includes/administration.inc.php

Package: Core\Administrator
Administrator
Administration functions of the ConnectME hypervideo annotation tool. Those are: * Check if a given URL responds and return the HTTP response * Parse the common log file and return its content * Parse the sparql query log and create a array out of it (with detail links to every query) * Parse a given file and return its content IMPORTANT: PEAR HTTP_Request library is used in this class (http://pear.php.net/package/HTTP_Request8)

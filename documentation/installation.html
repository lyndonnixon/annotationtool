Installation

Requirements

The ConnectMe annotation tool has been tested with PHP 5.3. Other PHP versions haven't been tested yet!

ConnectME Framework

You need a running instance of the ConnectME Framework (CMF) to use the annotation tool. The CMF is used as triple store and provides also several other function which are used by the annotation tool via webservices. Contact info@connectme.at to get access to a running instance or to get access to the installation files of the CMF.

PHP Libraries

Those PHP libaries are required for using the annotation tool

  • PEAR HTTP_Request; version 1.63; http://pear.php.net/package/HTTP_Request/
  • ARC2 Library; version 2011-12-01; https://github.com/semsol/arc2
  • Google APIs Client Library for PHP; http://code.google.com/p/google-api-php-client
  • Graphite PHP Linked Data Library; version 1.5; http://graphite.ecs.soton.ac.uk/
  • PHP OpenID library by JanRain; version 2.2.2; https://github.com/openid/php-openid
  • Smarty templating engine; version 3.1.12; http://www.smarty.net/

Directory structre

Create the following directory structure on your server and apply the specified read/write permissions to each directory. The annotation tool needs writing permissions in some directories!

 /
 /annotations/ => 0777
 /administrator/ => 0750
 /assets/ => 0750
 /assets/css/ => 0750
 /assets/img/ => 0750
 /assets/js/ => 0750
 /core/ => 0750
 /core/ajax/ => 0750
 /core/class/ => 0750
 /core/includes/ => 0750
 /core/lang/ => 0750
 /libraries/ => 0750
 /logs/ => 0777
 /plugins/ => 0750
 /plugins/dbpedia/ => 0750
 /plugins/geonames/ => 0750
 /plugins/maps_search/ => 0750
 /plugins/pre_annotations/ => 0750

Furthermore the files /core/includes/administration.inc.php and the files /logs/general.log and /logs/queries.log need the permission 0777!

The annotation tool source code already provides this structure, therefore you just have to set the proper read/write permissions after uploading it to your server.
